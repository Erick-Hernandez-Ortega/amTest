# amTest
## Instrucciones para ejecutar
- Tener node, de preferencia la versión 22
- Instalar los node modules 
```bash
npm i
```
- Despues ejecutar el json server
```bash
npm run json
```
- Despues ejecutar la app
```bash
npm run dev
```
